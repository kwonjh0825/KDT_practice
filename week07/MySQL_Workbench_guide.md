# Workbench를 활용한 MySQL 접속 방법

1. 터미널에서 `mysql.server start`를 통해 서버 실행
2. Workbench 실행
3. Local Instacne 클릭
4. Data Import -> Import from Self-Contained File 에서 원하는 파일 선택 후 Start Import
5. Schema 에서 데이터베이스가 import 되었는지 확인
